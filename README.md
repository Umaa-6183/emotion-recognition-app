# 🎭 Multi-Modal Emotion Recognition System

A real-time AI/ML-powered system that detects human emotions from facial expressions, voice tone, and text — using supervised learning and deep models.

> 🧠 Built for academic research, real-time applications, and human-computer interaction systems.

---

## 🚀 Features

- 🧍‍♀️ **Facial Emotion Detection** using webcam (FER + Deep Learning)
- 🎙️ **Voice Emotion Analysis** via microphone (Librosa + ML)
- 💬 **Text Sentiment Analysis** using Transformers
- 🔀 **Multi-Modal Fusion Engine** to combine predictions from all inputs
- 🧑‍💼 **Context-Aware Profiling** (age, gender, environment)
- 📊 Real-time visualizations in a Streamlit dashboard

---
